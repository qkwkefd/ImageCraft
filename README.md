# ImageCraft

以“轻量高效、功能实用、体验流畅”为核心定位，面向大众用户的图片编辑App。

项目说明：

### 1.基础技术栈：

- **开发语言**：Java 11

- **编译SDK版本**：36

- **最低支持Android版本**：28 (Android 9.0)

- **目标Android版本**：36

- **构建工具**：Gradle with Kotlin DSL

- **Android Gradle Plugin**：8.13.0

  

### 2.主要类结构

- **EditImageActivity**：应用主活动，管理整体界面和用户交互


- **CustomImageView**：自定义图片视图，实现图片编辑的核心功能

- **StickerElement**：贴纸元素类，管理贴纸的位置、缩放、旋转等属性

- **TextElement**：文字元素类，管理文字的位置、大小、颜色等属性

### 3. 功能模块

#### 3.1 图片导入与基础操作

- **图片导入**：支持从相册和相机拍摄导入图片
- **基础操作**：图片的拖动、缩放和平移
- **横竖屏支持**：自动适配屏幕方向变化

#### 3.2 图片编辑功能

##### 3.2.1 裁剪功能

- 自由裁剪区域选择
- 支持不同比例裁剪

##### 3.2.2 旋转功能

- 顺时针旋转90度
- 逆时针旋转90度
- 180度旋转

##### 3.2.3 调整功能

- 亮度调节
- 对比度调节

##### 3.2.4 滤镜功能

- 多种滤镜效果（原图、黑白、复古、清新等）

##### 3.2.5 文字编辑

- 添加文字
- 文字大小调整
- 文字颜色选择
- 字体选择
- 文字透明度调整
- 文字旋转和缩放

##### 3.2.6 贴纸功能

- 添加各种贴纸
- 贴纸缩放和旋转
- 贴纸透明度调整

#### 3.3 主题与设置

- **夜间模式切换**：支持深色/浅色主题切换
- **设置保存**：用户偏好设置本地保存
